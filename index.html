Responsive Paint

Trying to do a Paint webapp that adapts to mobile.

Download .zip Download .tar.gz View on GitHub

As a student, I followed a few class lessons about mobile adaptation. This simple Paint application is the result of these lessons.

You could try this app on Github

Interesting parts are:

  • desktop: as Firefox does not have any Color input elements, a basic but adaptive solution has been impemented ;
  • mobile: touch support and portrait/landscape adaption included, it also benefits the previous solution relative to the lack of support of Color input ;
